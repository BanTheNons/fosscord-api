# Fosscord Server
Consists of:
 * HTTP Rest API
 * CDN
 * Gateway
 * RTC

## Dev getting started
1. Install deps: `npm i`
2. TODO: Configure
3. Start the server:
 * To run all at once, run `npm start` in root directory.
 * To run api, do `npm start` in `api` folder
 * To run cdn, do `npm start` in `cdn` folder
 * To run gateway, do `npm start` in `gateway` folder
